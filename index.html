도움말

1. 사용설명서

데이터 로드

  • 이 앱은 kb023_information.xlsx, kb023_insurance.xlsx, kb023_prebargaining.xlsx, kb023_products.xlsx 파일을 같은 폴더에 두면 자동으로 로드합니다.
  • information 테이블은 페이지 로드 시 자동 표시됩니다. insurance/prebargaining은 information에서 행을 선택하면 해당 계약자 기준으로 표시됩니다.

DB고객 세부내용

  • 드롭다운(조회/입력/수정/삭제/출력)과 완료 버튼은 information에서 행을 선택한 뒤 사용할 수 있습니다. 수정·삭제·출력은 행 선택 후에만 선택 가능합니다.
  • 조회: 선택한 행의 세부내용·보험·사전교섭 데이터를 조회만 합니다.
  • 입력: 세부내용 폼을 비운 뒤 새로 입력합니다. 완료 버튼으로 저장(동작은 연동된 백엔드에 따라 다름).
  • 수정/삭제: 선택한 행 데이터를 수정하거나 삭제합니다. 완료 버튼으로 반영합니다.
  • 출력: 선택한 행 기준으로 DB고객 세부내용·information·insurance·prebargaining을 A4로 인쇄합니다. 인쇄 후 모드는 자동으로 조회로 돌아갑니다.

배정자료

  • 헤더의 배정자료 버튼을 누르면 information/insurance 데이터를 배정일자·계약자 순으로 모달에 표시합니다.
  • 모달은 마우스 드래그 또는 방향키로 이동할 수 있으며, 다음에 열 때 마지막 위치에서 표시됩니다.
  • 행을 클릭하면 모달이 닫히고 메인 화면에서 해당 information 행이 선택된 상태로 표시됩니다.

2. 주의사항

  • Excel 파일은 반드시 앱과 같은 경로(또는 지정된 경로)에 두어야 하며, 시트 첫 행은 헤더로 사용됩니다.
  • 주민등록번호·연락처 등 개인정보가 화면에 표시되므로, 사용 환경에 맞게 보안(접근 제한, 화면 보호)을 유지하세요.
  • 입력/수정/삭제 완료 시 실제 저장은 서버·로컬 연동 방식에 따릅니다. 저장 실패 시 데이터가 반영되지 않을 수 있습니다.
  • 인쇄 시 insurance/prebargaining 테이블은 내용이 길면 줄바꿈되며, information 등은 말줄임으로 표시될 수 있습니다.
